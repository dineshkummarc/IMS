IMS Documentation

Just follow these steps and you can easly install and use IMS in just few minutes

1. Install

NOTE: Your PHP Version should be >= 5.5.9 Just Follow these steps for install

1. Create Database in your phpmyadmin
2. Import Sql file, its located in sql folder
3. Change Database Detail in conn.php its located on root, and .env its located in local folder
Thats it. You are ready to use.

2. Configure Emails Details

1. Open .env file its located in local folder,Change your email detail in lines no 21 to 26
2. Open mail.php its located in local/config set sender name and address
Thats it.
